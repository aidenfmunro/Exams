# Билет 09. Теоремы Фубини и независимые случайные величины

Источники: Ширяев 1, гл. II.

## 1. Короткий ответ

Фубини позволяет считать интегралы по произведению мер повторно; независимость означает факторизацию совместного распределения. Главная формула: для $f\ge0$: $\int f\,d(\mu\otimes\nu)=\int(\int f(x,y)\,d\nu(y))\,d\mu(x)$. Если $X,Y$ независимы, то $P_{X,Y}=P_X\otimes P_Y$.

## 3. Основные определения

- произведение мер.
- Тонелли.
- Фубини.
- независимость событий, σ-алгебр и случайных величин.
- свертка распределений.

## 4. Основные теоремы и свойства

- для $f\in L^1$ повторные интегралы равны.
- независимость равна факторизации совместного закона.
- $\mathbb E[g(X)h(Y)]=\mathbb E g(X)\mathbb E h(Y)$.
- плотность независимого вектора равна произведению.

## 5. Примеры

- $p_{X,Y}=p_Xp_Y$.
- сумма независимых величин через свертку.
- двойной интеграл по прямоугольнику.

## 6. Схемы доказательств / обоснований

Доказать для индикаторов прямоугольников, затем для простых функций и предельным переходом; независимость переносится с порождающих классов.
