# Билет 25. Спектральные характеристики

Источники: Ширяев 2, гл. VI; Сердобольская, лекция 8.

## 1. Короткий ответ

Спектральная теория представляет корреляционную функцию стационарной последовательности как коэффициенты Фурье конечной меры. Главная формула: $R(k)=\int_{-\pi}^{\pi} e^{ik\lambda}F(d\lambda)$. Если есть плотность $f$, то $R(k)=\int e^{ik\lambda}f(\lambda)\,d\lambda$ с выбранной нормировкой.

Начать ответ нужно с роли темы: Спектральная теория представляет корреляционную функцию стационарной последовательности как коэффициенты Фурье конечной меры. После этого дать строгие определения, записать формулу и отдельно назвать условия, при которых она верна. В ДСП эта тема нужна не сама по себе, а как часть языка для распределений, случайных последовательностей, марковских цепей, корреляционных и спектральных характеристик.

Основная запись: $R(k)=\int_{-\pi}^{\pi} e^{ik\lambda}F(d\lambda)$. Если есть плотность $f$, то $R(k)=\int e^{ik\lambda}f(\lambda)\,d\lambda$ с выбранной нормировкой. ## 3. Основные определения

- спектральная мера.
- спектральная функция.
- спектральная плотность.
- ортогональная случайная мера.
- спектральное представление.

## 4. Основные теоремы и свойства

- корреляционная функция положительно определена.
- по Герглотцу ей соответствует конечная неотрицательная мера.
- атом спектра дает периодическую компоненту.
- абсолютно непрерывная часть описывает распределение мощности.

## 5. Примеры

- белый шум: плоский спектр.
- гармоника: атомы на $\pm\lambda_0$.
- $AR(1)$: рациональная спектральная плотность.

## 6. Схемы доказательств / обоснований

Использовать положительную определенность $R(k)$ и теорему Герглотца; процесс второго порядка представляется интегралом $X_n=\int e^{in\lambda}Z(d\lambda)$.
