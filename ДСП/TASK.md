# План работы над конспектами к экзамену по ДСП

## Цель

Сделать полный комплект конспектов по 25 билетам экзамена по дискретным случайным процессам так, чтобы по ним можно было спокойно подготовиться за 3 дня.

Итоговые материалы должны быть понятными, структурированными и пригодными для устного ответа на экзамене.

# 1. Что нужно сделать

Нужно подготовить конспекты по всем 25 билетам программы.

Для каждого билета нужно сделать отдельный файл:

```text
tickets/01.md
tickets/02.md
...
tickets/25.md
```

Каждый билет должен содержать:

1. полный устный ответ;
2. основные определения;
3. основные теоремы и свойства;
4. примеры;
5. связи с другими билетами через обсидиан ссылки;

---

# 2. Как писать конспекты

Писать не как учебник, а как материал для устного экзамена.

Каждый билет должен отвечать на вопросы:

1. Что это такое?
2. Зачем это нужно?
3. Какая главная формула?
4. Какой простой пример?
5. Как это доказывается в общих чертах?
6. С чем это связано?
7. Что может спросить преподаватель?

Стиль:

* русский язык;
* математически аккуратно;
* без лишней воды;
* определения формулировать строго;
* после строгой формулировки давать короткую интуицию.

# 4. Источники

Использовать материалы:

```text
Shiryaev_A_N_Veroyatnost_1.pdf
Shiryaev_A_N_Veroyatnost_2.pdf
theory-of-random-processes-lectures-serdobolskaya-M1.pdf
DSP_Lectures.pdf
```

Приоритет источников:

1. **ДСП_Лекции.pdf** — обозначения, стиль лектора, акценты курса.
2. **Ширяев 1** — основания вероятности, меры, интеграл Лебега, условное ожидание, характеристические функции, гауссовские системы, ЦПТ.
3. **Ширяев 2** — случайные последовательности, стационарность, эргодичность, спектральная теория, марковские цепи.
4. **Сердобольская** — случайные процессы, траектории, сечения, моменты, марковские процессы, теория второго порядка.

# 5. Формат каждого билета

Каждый билет писать по шаблону:

```markdown
# Билет XX. Название билета

## 1. Короткий ответ 

Кратко объяснить суть билета без деталей.

## 2. Полный ответ

Связный текст, который можно произнести на экзамене.

## 3. Основные определения

Дать все определения, которые относятся к билету.

## 4. Основные теоремы и свойства

Сформулировать главные утверждения.

## 5. Примеры

Дать 2–5 примеров.

## 6. Связи с другими билетами

К каким билетам эта тема относится.
```

# 8. Как проверять качество

После написания проверить:

## Полнота

* есть все 25 билетов;
* нет пустых разделов;
* есть дополнительные файлы.

## Единый стиль

Все билеты имеют одинаковую структуру.

## Математическая корректность

Особенно проверить:

* определения алгебры, полуалгебры и σ-алгебры;
* условия счётной аддитивности;
* формулу меры по функции распределения;
* условия теоремы Радона—Никодима;
* определение условного ожидания;
* формулы производных характеристической функции;
* отличие узкой и широкой стационарности;
* марковское свойство;
* уравнение Чепмена—Колмогорова;
* условия эргодичности;
* нормировки в спектральном представлении.

## Практичность

В каждом билете должно быть понятно:

* что сказать первым;
* что написать на доске;
* какой пример привести;
* какие допвопросы ожидать.
